Ο χρήστης θα μπορεί να δημιουργήσει έναν προσωπικό λογαριασμό και να συνδεθεί στην εφαρμογή, ώστε να οργανώνει και να παρακολουθεί τις αγαπημένες του ταινίες και σειρές. Μέσα από την εφαρμογή, θα έχει τη δυνατότητα να αναζητά συγκεκριμένους τίτλους και να τους προσθέτει στη λίστα του για μελλοντική προβολή. Όταν ολοκληρώνει την παρακολούθηση μιας ταινίας ή σειράς, θα μπορεί να τη σημειώνει ως "προβεβλημένη" και να λαμβάνει προτάσεις βασισμένες στις προτιμήσεις του. Παράλληλα, θα μπορεί να αξιολογεί το περιεχόμενο που έχει δει και να βλέπει στατιστικά στοιχεία για τις προβολές του, όπως το πόσες ώρες έχει αφιερώσει ή ποια είδη προτιμά περισσότερο.  

Η εφαρμογή θα του προσφέρει επίσης τη δυνατότητα να λαμβάνει ειδοποιήσεις για νέες κυκλοφορίες που μπορεί να τον ενδιαφέρουν. Επιπλέον, θα περιλαμβάνει διαδραστικό περιεχόμενο, όπως κουίζ, για να δοκιμάζει τις γνώσεις του σχετικά με τις ταινίες και τις σειρές που έχει παρακολουθήσει. Η εφαρμογή θα επιτρέπει επίσης στον χρήστη να προβάλλει το προφίλ του, το οποίο θα περιλαμβάνει στατιστικά στοιχεία σχετικά με τις προτιμήσεις του. Για παράδειγμα, θα μπορεί να βλέπει ποια κινηματογραφικά είδη έχει παρακολουθήσει περισσότερο, καθώς και τη συνολική διάρκεια προβολής του περιεχομένου που έχει δει. Επιπλέον, η εφαρμογή θα υποστηρίζει κοινόχρηστες λίστες προβολής (shared watchlists), δίνοντας τη δυνατότητα στον χρήστη να δημιουργεί και να μοιράζεται λίστες με άλλους χρήστες. Έτσι, θα μπορεί να ανακαλύπτει νέο περιεχόμενο μέσα από τις προτιμήσεις των φίλων του και να συντονίζει κοινές προβολές.
